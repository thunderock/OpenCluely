    Create `src/core/frame-dedup.js` — PURE CommonJS, zero Electron/fs imports, exporting exactly:
    - `grayscaleFromBgra(buffer, width, height)` → Uint8Array(width*height) luma. BGRA order (Electron toBitmap on macOS): `b=buf[i*4], g=buf[i*4+1], r=buf[i*4+2]`; luma = Math.round(0.299*r + 0.587*g + 0.114*b).
    - `dhash(luma, hashW = 16, hashH = 16)` → Buffer(hashW*hashH/8). Input luma is (hashW+1) x hashH row-major (17x16 = 272 px for defaults). Bit k (row-major x,y) = 1 when luma[y*(hashW+1)+x] > luma[y*(hashW+1)+x+1]. Pack MSB-first into a 32-byte Buffer.
    - `hamming(a, b)` → integer popcount of XOR over the two Buffers (throw TypeError if lengths differ — programmer error, not runtime input).
    - `blackStats(luma)` → `{ mean, variance }` over the luma array (plain number math, no sampling — 272 px is tiny).
    Keep functions dependency-free so `node --test` runs them under bare node. JSDoc each export with the BGRA/dimension contract.
