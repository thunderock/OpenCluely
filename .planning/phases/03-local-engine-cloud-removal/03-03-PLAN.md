  <action>
Create `LocalProvider extends LLMProvider` (from `./llm-provider`), CommonJS, exporting `{ LocalProvider }` (mirror `gemini.provider.js`'s export). Use `const OpenAI = require('openai')`, `const config = require('../../core/config')`, `const { RequestBuilder } = require('../../core/request-builder')`, and `const logger = require('../../core/logger').createServiceLogger('LOCAL')`. Logger contract: variable data goes in the meta object, never interpolated into the message.

**constructor()**: read `config.get('llm.local')`; set `this.host`, `this.model`, `this.keepAlive`. Instantiate `this.requestBuilder = new RequestBuilder()` (defaults to real singletons, like GeminiProvider). Call `this.initializeClient()`. Set `this.requestCount = 0; this.errorCount = 0`. MUST NOT throw if Ollama is down (constructing the openai client is network-free).

**initializeClient()**: `this.client = new OpenAI({ baseURL: \`${this.host}/v1\`, apiKey: 'ollama' })` (apiKey is required-but-ignored per RESEARCH Flag 5). Set `this.isInitialized = true`. Re-read model/host from config so a settings change picked up on relaunch takes effect.

**serialize(neutral)** — the ONLY place OpenAI wire shape is built (SC4). Implement exactly per RESEARCH Flag 2:
```js
serialize(neutral) {
  const messages = [];
  const sys = [neutral.systemPrompt, neutral.mdContext].filter(Boolean).join('\n\n'); // mdContext empty until Phase 5 — wire it now
  if (sys) messages.push({ role: 'system', content: sys });
  for (const h of neutral.history) {
    messages.push({ role: h.role === 'model' ? 'assistant' : 'user', content: h.content }); // neutral 'model' → OpenAI 'assistant'
  }
  if (neutral.images && neutral.images.length) {
    messages.push({ role: 'user', content: [
      { type: 'text', text: neutral.userText },
      ...neutral.images.map(i => ({ type: 'image_url', image_url: { url: `data:${i.mimeType};base64,${i.data}` } }))
    ]});
  } else {
    messages.push({ role: 'user', content: neutral.userText });
  }
  return { model: this.model, messages };
}
```
The image part MUST be the nested object `{ url: "data:<mime>;base64,<b64>" }` (RESEARCH Flag 2 — the bare-string variant is superseded). No skill/prompt/history assembly here — that stays in RequestBuilder.

**isAvailable()**: return `!!this.isInitialized` (client constructed). (Liveness of Ollama is a separate testConnection/health concern.)

**generate(neutral, options)**: `const { model, messages } = this.serialize(neutral); const res = await this.client.chat.completions.create({ model, messages, stream: false, keep_alive: this.keepAlive });` return the text (`res.choices[0].message.content`), applying `enforceProgrammingLanguage` when `options.programmingLanguage` (see Task 2). Pass `keep_alive` in the body as defense-in-depth (Ollama accepts it as an extension; the authoritative resident mechanism is LocalModelManager's warm-up in 03-04). Wrap in try/catch → on error, log + rethrow (callers handle fallback); never crash the process.

**generateStream(neutral, options, onDelta)**: serialize → `const stream = await this.client.chat.completions.create({ model, messages, stream: true, keep_alive: this.keepAlive });` then `for await (const chunk of stream) { const delta = chunk.choices?.[0]?.delta?.content || ''; if (delta) { full += delta; if (typeof onDelta === 'function') onDelta(delta); } }` return `full` (language-enforced when set). Match the RESEARCH "LocalProvider streaming via openai SDK" example.

**testConnection()**: trivial ping — try a 1-token `chat.completions.create` (or fetch `${host}/api/version`); return `{ success: true, model, ... }` on success or `{ success: false, error }` on failure. Never throw.
