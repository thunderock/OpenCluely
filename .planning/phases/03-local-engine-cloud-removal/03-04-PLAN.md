  ```js
  const ollamaHost = this.host.replace(/^https?:\/\//, ''); // '127.0.0.1:11434' for the daemon env
  const def = {
    name: 'ollama',
    command: this._resolveOllamaBin(),        // system `ollama` on PATH; guide-install if absent (Task returns a not-installed status, never throws)
    args: ['serve'],
    env: { OLLAMA_HOST: ollamaHost, OLLAMA_KEEP_ALIVE: String(this.keepAlive) }, // applied only when WE spawn
    healthCheck: { type: 'http', port: 11434, path: '/', timeoutMs: 1000 },
    backoff: { initialDelayMs: 500, multiplier: 2, maxDelayMs: 15000, maxRetries: 8 },
    startupTimeoutMs: 30000,
  ```

Methods:
- `async start()`: if Ollama binary not found → return `{ ok:false, reason:'not-installed' }` (drives the guide-install UX; do NOT spawn). Else `await this.supervisor.start()` (adopts if present, else spawns+health-polls). Return `this.getStatus()`.
- `async stop()`: `await this.supervisor.stop()` (no-op if adopted).
- `async detect()`: probe `${host}/api/version` (built-in `fetch`, timeout-bounded); return `{ running: boolean, installed: boolean }` (installed = binary resolvable on PATH).
- `async ensureModel(tag = this.model, { onProgress } = {})`: `const list = await this.ollama.list()`; if `tag` not in the installed models → `await this.pullModel(tag, { onProgress })`; then `await this.warmUp(tag)`. Return `{ ok:true, present:true }` (or an error struct on failure — never throw).
- `async pullModel(tag, { onProgress } = {})`: RESEARCH Flag 6 —
  ```js
  for await (const part of await this.ollama.pull({ model: tag, stream: true })) {
    const percent = part.completed && part.total ? Math.round(part.completed / part.total * 100) : null;
    if (typeof onProgress === 'function') onProgress({ status: part.status, percent, completed: part.completed, total: part.total });
  }
  return { ok: true, model: tag };
  ```
  `ollama pull` resumes interrupted layers + verifies sha256 before `success` (satisfies resumable/checksum — do NOT hand-roll range requests). Cache stays at Ollama's default (`~/.ollama/models`) — do NOT set a custom cache path.
- `async warmUp(tag = this.model)`: enforce resident behavior regardless of adopt/own with a one-shot `await this.ollama.generate({ model: tag, prompt: '', keep_alive: -1 })` (RESEARCH Open Q1 recommendation). Belt-and-suspenders with the serve env above. Swallow errors (best-effort).
- `async listInstalledModels()`: `return (await this.ollama.list()).models.map(m => m.name)` — for the advanced "any installed" picker (03-05).
- `async getStatus()`: fuse `this.supervisor.getStatus()` (`{ state, owned, pid }`) with three-level health (RESEARCH Flag 3 / Pitfall 4): (1) server up (`/api/version`), (2) model present (`list()` contains model), (3) model responds (a trivial generate). Return `{ owned, adopted: state==='adopted', state, serverUp, modelPresent, modelResponds, model }` — each level a distinct field so the UI can give three distinct messages.
- `async preflight()`: check free disk on the Ollama cache volume (use `fs.statfs` if available, else best-effort) needs ≳ 7 GB; check `require('os').totalmem()` and WARN (do NOT block) if `< 16 GB` (qwen3-vl:8b recommended floor). Return `{ ok, diskOk, ramGb, warnings: [...] }`. Friendly failure per the locked decision.
- `_resolveOllamaBin()`: resolve `ollama` on PATH (use `which`/`where` via `spawnSync`, mirroring whisper-installer `_detectPython`); return the path or null.

Do NOT add `tree-kill`/`get-port`/`execa` (RESEARCH: supervisor SIGTERM→SIGKILL on the owned child is the locked mechanism; Ollama's port is the fixed 11434).
