  <action>
Create `src/core/service-supervisor.js` (src/core style: 2-space, single quotes). Follow the RESEARCH "ServiceSupervisor" code example (01-RESEARCH.md) for the two health probes VERBATIM — they are the HIGH-confidence part — and follow the DI/export shape of `WhisperInstaller` (export the class; `this.spawn = options.spawn || spawn`; also export the probe helpers). The RESEARCH lifecycle sketch is a DESIGN, not verified code — implement the HARDENED control model below instead, which fixes the double-restart / hang risks (Pitfall 6).

Requires: `const { spawn } = require('child_process'); const net = require('net'); const http = require('http'); const EventEmitter = require('events'); const logger = require('./logger').createServiceLogger('SUPERVISOR');`

Pure helpers (export all):
- `probePort({ host = '127.0.0.1', port, timeoutMs = 1000 })` -> Promise<boolean>: `net.connect`, `socket.setTimeout(timeoutMs)`, resolve true on `'connect'`, false on `'timeout'`/`'error'`, always `socket.destroy()`; guard against double-resolve. (VERBATIM from RESEARCH.)
- `probeHttp({ host = '127.0.0.1', port, path = '/', timeoutMs = 1000 })` -> Promise<boolean>: `http.get({host,port,path,timeout})`, `res.resume()` then resolve `res.statusCode > 0`; on `'timeout'` destroy+false; on `'error'` false. (VERBATIM from RESEARCH.)
- `computeBackoffDelay(attempt, backoff)` -> number: `Math.min(backoff.maxDelayMs, backoff.initialDelayMs * (backoff.multiplier ** attempt))`. Pure and separately unit-testable.

`class ServiceSupervisor extends EventEmitter`:
- `constructor(definition, options = {})`: store `this.def = definition`; `this.spawn = options.spawn || spawn`; `this.logger = options.logger || logger`; init `this.child=null; this.owned=false; this.state='idle'; this.attempt=0; this._intentionalStop=false; this._backoffTimer=null; this._startupSettled=false;`.
  - `definition = { name, command, args=[], cwd?, env?, healthCheck, backoff, startupTimeoutMs?, healthPollMs?, adopt?, pidFile?, terminate? }`
  - `healthCheck = { type: 'port'|'http', host?, port, path?, timeoutMs? }`
  - `backoff = { initialDelayMs, multiplier, maxDelayMs, maxRetries }`
  - `terminate = { sigtermGraceMs }`
- `getStatus()` returns `{ name, state, attempt, pid: this.child ? this.child.pid : null, owned }`.
- `_setState(state, extra={})`: set `this.state`, `this.emit('status', this.getStatus())`, `this.logger.info('supervisor state', { state, ...extra })`. NEVER string-interpolate variable data into the message (logger contract).
- `_probe()`: `return this.def.healthCheck.type === 'http' ? probeHttp(hc) : probePort(hc);`
- `_waitHealthy(timeoutMs)`: poll loop until `Date.now() >= deadline`; each iteration: if `this._startupSettled` return false; if `await this._probe()` return true; else `await sleep(this.def.healthPollMs || 50)`. Returns false on timeout.

HARDENED lifecycle (this is the load-bearing part — get the guards right):
- `async start()`: `this._intentionalStop = false;` then ADOPT-IF-PRESENT — if `this.def.adopt && await this._probe()`: `this.owned = false; this._setState('adopted'); return this.getStatus();` (stop() must never kill this). Otherwise `return this._attemptStart();` (OWN-IF-STARTED).
- `async _attemptStart()`:
  1. `this._startupSettled = false; this._setState('starting', { attempt: this.attempt });`
  2. `try { this.child = this.spawn(this.def.command, this.def.args || [], { cwd, env: env ? { ...process.env, ...env } : process.env, windowsHide: true }); } catch (e) { return this._handleStartupFailure(e.message); }`
  3. `this.owned = true;` write the PID sidecar if `this.def.pidFile` (wrap in try/catch).
  4. Attach ONE-SHOT startup listeners that mark settled + route to `_handleStartupFailure`:
     `const onStartupExit = (code, signal) => { if (this._startupSettled) return; this._startupSettled = true; this._handleStartupFailure('exited during startup', { code, signal }); };`
     `this.child.once('error', (err) => { if (this._startupSettled) return; this._startupSettled = true; this._handleStartupFailure(err.message); });`
     `this.child.once('exit', onStartupExit);`
  5. `const healthy = await this._waitHealthy(this.def.startupTimeoutMs || 30000);`
  6. If `this._startupSettled` (startup exit/error already scheduled the restart): `return this.getStatus();`
  7. `this._startupSettled = true; this.child.removeListener('exit', onStartupExit);`
  8. If `!healthy`: `return this._handleStartupFailure('health check did not pass within startup timeout');`
  9. Healthy: `this.attempt = 0;` attach the LONG-RUNNING crash monitor `this.child.once('exit', (code, signal) => { if (this._intentionalStop) return; this.logger.warn('managed process exited', { code, signal }); this._scheduleRestart('crashed'); });` then `this._setState('healthy', { pid: this.child.pid }); return this.getStatus();`
- `_handleStartupFailure(reason, extra={})`: `this.logger.error('managed process failed to start', { reason, ...extra });` best-effort kill the failed child (`try { if (this.child) this.child.kill('SIGKILL'); } catch (_) {}`); `this._scheduleRestart(reason); return this.getStatus();`
- `_scheduleRestart(reason)`: if `this._intentionalStop` return; if `this._backoffTimer` return (guard against double-scheduling); if `this.attempt >= this.def.backoff.maxRetries` -> `this._setState('failed', { reason }); return;` (give up + surface). Else `const delay = computeBackoffDelay(this.attempt, this.def.backoff); this.attempt += 1; this._setState('restarting', { attempt: this.attempt, delayMs: delay, reason }); this._backoffTimer = setTimeout(() => { this._backoffTimer = null; if (!this._intentionalStop) this._attemptStart(); }, delay);`
- `async stop()`: `this._intentionalStop = true;` clear `this._backoffTimer` (and null it). If `!this.owned || !this.child` -> `this._setState('stopped'); return;` (NEVER kill an adopted/foreign process — SC4). Else capture `const child = this.child; const grace = (this.def.terminate && this.def.terminate.sigtermGraceMs) || 5000;` `child.kill('SIGTERM');` await `new Promise(res => { const t = setTimeout(() => res(false), grace); child.once('exit', () => { clearTimeout(t); res(true); }); })`; if not exited `child.kill('SIGKILL');`. Remove the PID sidecar if configured (try/catch). `this._setState('stopped');`

`module.exports = ServiceSupervisor; module.exports.ServiceSupervisor = ServiceSupervisor; module.exports.probePort = probePort; module.exports.probeHttp = probeHttp; module.exports.computeBackoffDelay = computeBackoffDelay;`

Design check (comment or SUMMARY note, do NOT wire real services — deferred to P3/P4): confirm both future consumers fit this contract unchanged — Ollama `{ healthCheck: { type:'http', port: 11434, path:'/' }, adopt: true }` and whisper-server `{ healthCheck: { type:'port', port }, adopt: false, pidFile, terminate: { sigtermGraceMs: 5000 } }`, per the RESEARCH "Both future consumers fit this contract" example.
